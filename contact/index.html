Get in touch!

There are lots of ways to get in touch. If you need any assistance with building or using Mirobot, then the best place for this is on the Mirobot Google Group

If you have any feedback on your experiences building Mirobot, please send it via the feedback survey

If you have a bug to report, please use the following places to report it:

  • PCB Issues
  • Chassis Issues
  • Software Issues
  • Arduino Issues

If you'd like to stay up to date on what's happening in the world of Mirobot, then please join the mailing list.

If you have any other questions, then please get in touch via email: info@mirobot.io
