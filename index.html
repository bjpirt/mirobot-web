Open Source

Mirobot is 100% open source, including:

  • PCB designs and schematics
  • Firmware code
  • Chassis designs

It's designed with learning in mind - take the files, learn how to build things and make your own!

Find out more >

Programmable

There are lots of ways to program Mirobot:

  • Easy visual programming language
  • In-browser Javascript
  • Directly with LOGO
  • Using the Arduino that's on board

Find out more >

Build it Yourself

Mirobot has been designed to be easily built from scratch

  • Assemble the chassis
  • Solder the circuit board
  • Program the Arduino

By building it yourself you build up more of an understanding of how it works
