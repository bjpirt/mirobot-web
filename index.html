Mirobot is a WiFi robotics kit that helps children learn about technology and programming

Get Started >

Mirobot is designed to make learning about technology easy and fun. Once you've built it yourself, you can program it through your web browser or via any other programming language.

Easy to build!

Mirobot has been designed to be easily built from scratch

  • Comes as a kit
  • Assemble the chassis
  • No soldering required!

By building it yourself you build up more of an understanding of how it works

Find out more >

Programmable

Use the apps to program Mirobot or start coding:

  • Easy visual programming language
  • In-browser Javascript
  • Using the on-board Arduino

There are lots of levels available to suit all types of learners.

Find out more >

It's Open!

It's easy to modify Mirobot if that's what you'd like

  • Customise the chassis
  • Change the Arduino code
  • Add on extra hardware

It's designed with learning in mind - take the files, learn how to build things and start customising!
